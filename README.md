This is a comprehensive web site of a blog. This site includes front end posting pages and back end administration pages. In the back end pages, user can log in and out, add/edit/delete post categories, add/manage posts; the front pages show the post list and details of each post based on request. This web site is created with Node.js, express, ejs. Data is stored in MongoDB. More details will be finished up.
